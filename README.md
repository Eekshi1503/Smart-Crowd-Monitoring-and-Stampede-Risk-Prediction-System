# Smart Crowd Monitoring & Stampede Risk Prediction System

## Overview
This project is a real-time crowd monitoring system designed to detect high-density areas and predict potential stampede risks using Computer Vision techniques.

##  Features
- Real-time human detection using YOLOv8
- Crowd density estimation
- Risk prediction for overcrowded areas
- Supports image, video, and live camera input
- Visual alerts for high-density zones

## Tech Stack
- Python
- OpenCV
- YOLOv8
- NumPy

##  How It Works
1. Capture video/live feed
2. Detect people using YOLO
3. Count number of people
4. Calculate density
5. Trigger alert if threshold exceeded

##  Applications
- Railway stations
- Public events
- Smart city surveillance
- Disaster management
